# SAP for FMCG Offline Learning Studio

An offline interactive learning page for understanding SAP S/4 module relationships in FMCG workflows.

## What this project includes

- A single-file offline learning app: `sap-fmcg-module0.html`
- Chinese, English, and German UI switching
- Interactive SAP terminology popups
- SAP GUI style training screens for MM, supply chain, finance, and customer flows
- Local-only demo data with no API, no login, and no company privacy exposure

## Open locally

Open the HTML file directly in a browser:

```bash
open sap-fmcg-module0.html
```

Or double-click the file in Finder.

## Safety

- No backend
- No API calls
- No company system connection
- No real SAP data
- All progress is stored only in browser local storage

## Main file

- `sap-fmcg-module0.html`: the full offline experience
